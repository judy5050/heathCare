# SAFE HIP

#### SAFE HIP은 변기 위생 관리 HW와 연동되는 앱으로 변기 사용시간 관리 어플리케이션입니다.
#### 변기 HW 정보 확인, 변기 자동 청소 기능을 제공합니다.
#### SAFE HIP을 통해 변기 사용시간을 감지해서 건강 정보 전달을 받을 수 있습니다.
#### 추가로 게시판 기능을 통해 익명으로 건강 정보를 공유할 수 있습니다.
